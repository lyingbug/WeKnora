# 使用场景

## 内部知识助手

上传产品文档、操作手册、制度、工单和 Runbook，让用户通过 Web UI 或 IM 获得带引用的回答。

## 企业 RAG 平台

把 WeKnora 作为私有化 RAG 层，灵活配置模型、向量库、对象存储和检索策略。

## 文档到 Wiki

使用 Wiki 模式把长文档转化为更适合浏览、维护和推理的 Markdown 知识网络。

## Agent 工具编排

连接 MCP 服务、网络搜索和自定义 Skill，让 Agent 执行受控的多步任务。

## 开发集成

通过 REST API、Go Client、CLI 或 MCP Server 把 WeKnora 集成到应用和自动化流程中。
