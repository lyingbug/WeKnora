# Extension Points

WeKnora is designed around replaceable providers and connectors.

## Common extension areas

- Model providers
- Embedding providers
- Rerank providers
- Vector stores
- Object storage backends
- Data source connectors
- IM connectors
- Web search providers
- MCP services
- Agent Skills

## Extension principles

Keep extensions isolated behind provider interfaces, store credentials securely, expose clear configuration, and include operational diagnostics for failures.
