# Retrieval Quality

Poor retrieval quality usually comes from mixed knowledge domains, weak parsing, unsuitable chunking, missing metadata, or model mismatch.

## Improvements

- Split unrelated content into separate knowledge bases.
- Review extracted text and chunks.
- Tune chunking strategy.
- Enable rerank for larger candidate sets.
- Use an embedding model appropriate for the document language.
