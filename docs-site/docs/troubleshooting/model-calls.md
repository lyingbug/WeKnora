# Model Call Issues

Model call failures can come from credentials, endpoint configuration, quota limits, unsupported parameters, or network restrictions.

## Checklist

- Verify provider credentials.
- Confirm endpoint and model names.
- Check network access from the backend service.
- Review provider quota and rate limits.
- Test chat and embedding models separately.
