# Ingestion Issues

Common ingestion problems include unsupported files, parser failures, embedding errors, and storage failures.

## Checklist

- Confirm the file format is supported.
- Check DocReader logs.
- Verify object storage access.
- Verify embedding model configuration.
- Inspect task status and retry failed jobs when appropriate.
