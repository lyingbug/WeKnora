# FAQ

## The Web UI does not load

Check that Docker Compose services are running and that the frontend or reverse proxy port is not occupied.

## Answers do not cite sources

Confirm that the selected knowledge base contains parsed chunks and that retrieval returns relevant results.

## Agent tools are not called

Check Agent configuration, MCP service status, tool permissions, and approval settings.
