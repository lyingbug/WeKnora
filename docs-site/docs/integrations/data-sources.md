# Data Sources

Data source connectors synchronize content from external systems into WeKnora knowledge bases.

## Typical connector responsibilities

- Authenticate with the external system.
- List available spaces, pages, or documents.
- Fetch content and metadata.
- Support incremental synchronization when possible.
- Report synchronization errors clearly.

## Supported direction

WeKnora is designed to support connectors such as Feishu, Notion, Yuque, and custom enterprise systems.
