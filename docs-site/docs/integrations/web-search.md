# Web Search

Web search providers allow Agents to retrieve fresh external information when local knowledge is insufficient.

## Common providers

- DuckDuckGo
- Bing
- Google
- Tavily
- Baidu
- SearXNG

## Guidance

Use web search for tasks that require external or recent information. For private knowledge workflows, prefer knowledge base retrieval and disable web search where external access is not allowed.
