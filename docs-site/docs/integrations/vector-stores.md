# Vector Stores

Vector stores power semantic retrieval over document chunks.

## Supported backend types

WeKnora is designed to work with PostgreSQL pgvector, Elasticsearch, OpenSearch, Milvus, Weaviate, Qdrant, Apache Doris, Tencent Cloud VectorDB, and custom vector store implementations.

## Selection criteria

Choose a vector store based on scale, operational familiarity, filtering needs, hybrid search support, and deployment constraints.
