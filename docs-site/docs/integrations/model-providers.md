# Model Providers

WeKnora can connect to multiple model providers for different model roles.

## Provider roles

- Chat generation
- Agent reasoning
- Embedding
- Rerank
- Vision-language understanding
- Speech recognition

## Configuration guidance

Keep model credentials scoped, use tenant-level controls where needed, and test each model role before enabling production workflows.
