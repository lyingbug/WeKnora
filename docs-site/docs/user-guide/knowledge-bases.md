# Knowledge Bases

A knowledge base is the main container for searchable knowledge in WeKnora.

## Supported content

- Uploaded files
- Imported URLs
- Markdown content
- FAQ entries
- Synchronized data sources
- Generated Wiki pages

## Common operations

- Create a knowledge base for a project, team, product, or domain.
- Upload documents and monitor parsing progress.
- Tag and organize knowledge items.
- Reparse documents when parsing settings change.
- Share knowledge bases across workspace members according to RBAC permissions.

## Design guidance

Prefer separating knowledge bases by access boundary and answer domain. A focused knowledge base usually produces better retrieval quality than one mixed with unrelated content.
