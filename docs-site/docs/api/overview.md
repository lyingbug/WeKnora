# API Overview

The WeKnora API lets applications manage knowledge bases, upload knowledge, search content, create chat sessions, send messages, manage Agents, and configure integrations.

## API consumers

- Internal business applications
- Automation scripts
- CLI tools
- MCP services
- Custom user interfaces

## Reference source

The repository contains Swagger output under `docs/swagger.yaml` and topic-based API notes under `docs/api/`. This documentation site should eventually publish the generated API reference directly.
