# Knowledge Base API

Knowledge base APIs are used to create, list, update, share, and delete knowledge bases.

Common workflows include:

- Creating a knowledge base for a project.
- Listing knowledge bases available to a user.
- Updating knowledge base metadata.
- Assigning models or retrieval settings.
- Managing access according to tenant permissions.
