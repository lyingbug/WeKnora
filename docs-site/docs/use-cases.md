# Use Cases

## Internal knowledge assistant

Upload product docs, operation manuals, policies, support tickets, and runbooks. Users ask questions through the Web UI or IM channels and receive answers with source citations.

## Enterprise RAG platform

Use WeKnora as a private RAG layer with configurable model providers, vector stores, object storage, and retrieval strategies.

## Document-to-Wiki workflow

Use Wiki mode to transform long documents into linked Markdown pages that are easier to browse, maintain, and reason over.

## Agent tool orchestration

Connect MCP services, web search, and custom skills so Agents can perform controlled multi-step tasks.

## Developer integration

Use the REST API, Go client, CLI, or MCP server to integrate WeKnora into applications and automation workflows.
