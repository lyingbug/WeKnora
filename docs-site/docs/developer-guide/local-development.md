# Local Development

Use local development mode when changing backend or frontend code frequently.

```bash
make dev-start
make dev-app
make dev-frontend
```

The frontend development server usually runs at `http://localhost:5173`, and the backend API usually runs at `http://localhost:8080`.

For backend hot reload, use Air if it is installed in your environment.
