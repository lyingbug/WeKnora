# Project Structure

Important repository directories include:

- `frontend/`: Web UI.
- `internal/`: Backend application code.
- `docreader/`: Document parsing service.
- `mcp-server/`: MCP server implementation.
- `cli/`: WeKnora CLI.
- `client/`: Go client.
- `helm/`: Helm chart.
- `docker/`: Docker image definitions.
- `docs-site/`: Public documentation site.
