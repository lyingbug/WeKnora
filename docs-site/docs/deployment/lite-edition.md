# Lite Edition

WeKnora Lite is intended for lightweight local or single-node usage.

Use Lite when you need:

- A simpler local experience.
- A compact deployment for demos or evaluation.
- Fewer external service dependencies.

Use the standard deployment when you need multi-user operations, stronger scalability, full service composition, or production-grade observability.
